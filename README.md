# Crypto AI Advisor & Educational Dashboard

Премиальное веб-приложение для новичков в сфере криптовалют. Предоставляет интерактивный дашборд с графиками TradingView, актуальными котировками, новостями рынка и безопасным ИИ-консультантом по техническому и фундаментальному анализу (на базе Google ADK / Gemini 2.5).

---

## 🛠️ Особенности безопасности (Курс Kaggle)
1. **Защита от инъекций (Prompt Shield)**: Встроенный валидатор проверяет входящие сообщения на попытки джейлбрейка, обхода правил и запуска нежелательных команд (таких как `sudo`, `rm` и т.д.).
2. **Confinement (Ограничение контекста)**: ИИ-консультант привязан к конкретному активу (BTC, ETH и т.д.). Попытки обсудить другие активы блокируются с предложением переключить монету в меню.
3. **Раздельные сессии**: Дашборд сохраняет и восстанавливает историю чата отдельно для каждой криптовалюты при переключениях.
4. **Запрет на финансовые сигналы**: Системная инструкция строго запрещает давать советы по покупке/продаже. ИИ выступает в качестве обучающего наставника.

## 🚀 Локальный запуск

1. Перейдите в папку проекта:
   ```bash
   cd f:\AGA\crypto-advisor
   ```

2. Убедитесь, что настроен файл `.env`. Он должен содержать ваш ключ Google AI Studio:
   ```env
   GEMINI_API_KEY=Ваш_Ключ_Из_AI_Studio
   GOOGLE_GENAI_USE_VERTEXAI=False
   ```

3. Запустите локальный веб-сервер FastAPI:
   ```bash
   uv run uvicorn app.fast_api_app:app --reload --port 8000
   ```

4. Откройте браузер по адресу: [http://localhost:8000](http://localhost:8000)

## ☁️ Развертывание в Google Cloud Run

Чтобы развернуть проект в облаке Google Cloud:

1. Убедитесь, что вы авторизованы в gcloud CLI и настроили ваш проект:
   ```bash
   gcloud auth login
   gcloud config set project project-9e1f8f1a-77b4-471a-b1e
   ```

2. Соберите Docker-образ и разверните его на Cloud Run одной командой:
   ```bash
   gcloud run deploy crypto-advisor \
       --source . \
       --region us-central1 \
       --allow-unauthenticated \
       --set-env-vars="GEMINI_API_KEY=YOUR_GEMINI_API_KEY_HERE,GOOGLE_GENAI_USE_VERTEXAI=False"
   ```

После успешного завершения команды gcloud выдаст вам публичную ссылку на работающий сайт!
